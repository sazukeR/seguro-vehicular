## How to use it


1. Clone the repository
```
git clone https://github.com/sazukeR/seguro-vehicular.git
```

2. Move to project folder
```
cd seguro-vehicular
```

3. Install project dependencies
```
npm install
```

4. Run the project
```
npm run dev
```
